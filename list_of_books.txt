Genre Id's: Fantasy = fanISBN, Horror = horrISBN, Dystopian = dystISBN, Gothic = gothISBN, 
Sceince Fiction = ScifiISBN, Mythology = mythISBN, Adventure Fiction = AdvfiISBN, Mystery = mystISBN

Example Queries:

SELECT *
	FROM Books
	JOIN BelongsTo USING (isbn)
	JOIN Genre USING (g_id);
	
SELECT Books.isbn AS ISBN, Books.b_name AS Name, Books.ch_ct AS Chapter_Count, Books.release_date, Books.author_name, Genre.g_name AS Genre
	FROM Books
	JOIN BelongsTo USING (isbn)
	JOIN Genre USING (g_id);
